Github and Atom

What is GitHub?

GitHub is a website that houses repositories and to a lesser degree Pages.

What is Atom?

Atom is a code editor that is created by the same developers who created GitHub.

What is the relationship between the two?

GitHub and Atom can be used together to optimize your efficiancy while coding.

    It is easy to set them up to run together:
  1. First download Atom. This is where you do your coding.
  2. Next, make a GitHub account.
  3. After that, download GitHub for DeskTop. Us your same login for GitHub to sign into GitHub for DeskTop.
  4. Go back to your GitHub account and create a respotory, it should be yourusername.github.io. You can do this by clicking "Start a Project" on your homepage.
  5. Now head back to GitHub for DeskTop and click the "+" sign in the top left corner. You want to click the clone tab and make a clone of your github.io. It will appear on the left side of your GitHub for Desktop application.
  6. Right click your repo and select "Open in Atom"
  7. From here, any changes you make to your code will show up on the GitHub for DeskTop application.
  8. Once you're done making changes, you can save your html document and go to the GitHub for DeskTop application.
  9. At the top of the page there will be a tab that says "1 Uncommited Change." Click that.
  10. This will bring you to a page that will show all your code. Write something in the summary field and click "Commit to master."
  11. After this, click "Sync." Your website will now be uploaded online.
